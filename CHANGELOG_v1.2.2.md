# Version 1.2.2 (2026-07-23)

Version 1.2.2 is a presentation-only maintenance release. It preserves the corrected v1.2.1 numerical workflow and all scientific source values.

## Changes

- Added modified Knapp–Hartung 95% confidence intervals and Holm-adjusted P values to the manuscript-facing Table 2 for the eight diagnostic signature–window contrasts.
- Removed the threshold-dependent `T48 pattern` column from Table 2.
- Updated the Table 2 notes to distinguish four changes supported by both the standard and modified procedures from two inference-sensitive estimates.
- Updated release documentation and citation metadata to version 1.2.2 and DOI `10.5281/zenodo.21506111`.

## Unchanged scientific content

- Cohorts, analysis populations, effect estimates, standard Hartung–Knapp results, prediction intervals, heterogeneity estimates, and gene-contribution metrics.
- Figures 1–6, Supplementary Figures S1–S5, and all numerical figure-source values.
- The corrected v1.2.1 analysis and reproducibility workflow, whose internal versioned paths and filenames are retained.

The prior v1.2.1 release remains part of the immutable version history.
