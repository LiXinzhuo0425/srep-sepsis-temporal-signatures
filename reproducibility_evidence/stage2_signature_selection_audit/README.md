# Stage 2 signature-selection audit

This directory contains the frozen eligibility criteria, candidate registry, and reproducibility grading used to define the evaluated signature set.

- Candidate discovery used a structured concept search completed on 15 July 2026, supplemented by review-assisted candidate-name finding.
- Formula, gene membership, and implementation details were checked against primary publications or official sources where available.
- The frozen registry contains 34 candidates: A1 = 0, A2 = 10, B = 19, and X = 5.
- Eight protein-coding A2 signatures were included because they could be reconstructed across all six primary cohorts without refitting.
- Two additional A2 candidates were lncRNA signatures and were not evaluated because cross-platform feature mapping was unresolved.
- This was a structured source and reproducibility audit, not an exhaustive systematic review or catalogue of all published sepsis transcriptomic signatures.
- Selection was based on prespecified reconstructability and scope criteria, not on longitudinal effect size, direction, significance, or any other study result.

The three frozen source files in this directory retain their original filenames and content hashes.
